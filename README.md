#insulin_pump_system
Dual microcontroller insulin pump system for monitoring glucose levels and delivering insulin
- One to monitor glucose levels from photoresistor
- One to deliver insulin

#Features
- Real time glucose sampling
- Threshold based insulin delivery
- Serial communication between microcontrollers
- Safety cutoff

#Tech Used
- C language
- UART serial communication
- Microcontrollers (MSP432P4111)
- Hardware prototyping
